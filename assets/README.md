# TrustMatch App Icons - Base64 Encoded PNG Files

## Instructions to decode these files:
1. Copy the base64 content from each .base64 file
2. Go to https://base64.guru/converter/decode/image
3. Paste the base64 data and download as PNG
4. Rename the downloaded files to: icon.png, splash.png, adaptive-icon.png, favicon.png
5. Place them in your TrustMatch/assets/ folder
